- Prefix: Ctrl + B

# Pannels

## Horizontal split

```
    <Prefix> + %
```

## Vertical split

```
    <Prefix> + "
```

## Move cursor

```
    <Prefix> + <arrow-key>
```

## Unzoom/Zoom pannel

```
    <Prefix> + Z
```

## Close pannel

```
    Ctrl + D
```

# Windows

## Create window

```
    <Prefix> + C
```

## Switch window

```
    <Prefix> + P # Move to previous
    <Prefix> + N # Move to next
```

## Rename window

```
    <Prefix> + ,
```

## Close window

```
    Ctrl + D
```

# Sessions

## List sessions

```bash
    tmux ls
```

## Attach sessions

```bash
    tmux a -t <session-name>
```

## Detach sessions

```
    <Prefix> + D
```

## Rename session

```
    <Prefix> + $
```

## Switch sessions

```
    <Prefix> + S
```
